02.

BOOK BLOG

I developed a book blog using Node.js, Express, EJS, and PostgreSQL, allowing users to log and review books. The app stores book details in a SQL database and dynamically renders content using EJS templates. To enhance the experience, I integrated the OpenLibrary API to fetch book covers. The site features a responsive design for smooth navigation across devices, and JavaScript is used to create an interactive and user-friendly interface.

03.

TO DO LIST

I built a to-do list app using Node.js, Express, EJS, and PostgreSQL, implementing a many-to-many relationship to efficiently manage tasks and categories. Users can create, organize, and track tasks, with each task able to belong to multiple categories. The app dynamically renders content using EJS templates, ensuring a seamless and interactive user experience. With a responsive design, it adapts to different screen sizes, making task management easy across devices.

04.

SHOPIFY WEBSITE

I developed a Shopify website for Pippa Smith Studio, an independent art and homeware brand. The site features a clean, visually appealing design that showcases handcrafted products with high-quality imagery and intuitive navigation. I customized the Shopify theme using Liquid, HTML, and CSS to align with the brand's aesthetic and ensure a seamless shopping experience. The site is fully responsive, providing smooth functionality across devices, and includes e-commerce features such as secure checkout, product filtering, and customer account management.
