BOOK BLOG

I developed a book blog using Node.js, Express, EJS, and PostgreSQL, allowing users to log and review books. The app stores book details in a SQL database and dynamically renders content using EJS templates. To enhance the experience, I integrated the OpenLibrary API to fetch book covers. The site features a responsive design for smooth navigation across devices, and JavaScript is used to create an interactive and user-friendly interface.
